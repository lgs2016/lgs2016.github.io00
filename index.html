数组 创建数组

数组是一个固定长度的，包含了相同类型数据的 容器
步骤 1 : 声明数组
int[] a; 声明了一个数组变量。
[]表示该变量是一个数组
int 表示数组里的每一个元素都是一个整数
a 是变量名
但是，仅仅是这一句声明，不会创建数组

步骤 2 : 创建数组
创建数组的时候，要指明数组的长度。
new int[5]
引用概念：
如果变量代表一个数组，比如a,我们把a叫做引用
与基本类型不同
int c = 5; 这叫给c赋值为5
声明一个引用 int[] a;
a = new int[5];
让a这个引用，指向数组

步骤 3 : 访问数组
数组下标基0
下标0，代表数组里的第一个数

步骤 4 : 数组长度
.length属性用于访问一个数组的长度
数组访问下标范围是0到长度-1
一旦超过这个范围,就会产生数组下标越界异常

步骤 5 : 练习-数组最小值
步骤 5 : 练习-数组最小值

然后给数组的每一位赋予随机整数
通过for循环，遍历数组，找出最小的一个值出来
0-100的 随机整数的获取办法有多种，下面是参考办法之一:
(int) (Math.random() * 100)
Math.random() 会得到一个0-1之间的随机浮点数，然后乘以100，并强转为整型即可。

Arrays是针对数组的工具类，可以进行 排序，查找，复制填充等功能。 大大提高了开发人员的工作效率。

equals 判断是否相同

步骤 1 : 数组复制 

import java.util.Arrays;

public class HelloWorld{
    public static void main(String[] args) {
        int a[]=new int[]{34,55,45,435,36};
        int[]b=Arrays.copyOfRange(a, 0, 3);
        for(int i:b){
        System.out.println(i);
        }
    }
}

步骤 3 : 排序   
```java
import java.util.Arrays;
public class HelloWorld{
    public static void main(String[] args) {
        int a[]=new int[]{34,46,45,43,37};
        System.out.println("排序之前：");
        System.out.println(Arrays.toString(a));
        Arrays.sort(a);
        System.out.println("排序之后：");
        System.out.println(Arrays.toString(a));
    }
}
```java
步骤 4 : 搜索 
```java  
import java.util.Arrays;
public class HelloWorld{
    public static void main(String[] args) {
        int a[]=new int[]{34,46,45,43,37};
        System.out.println("排序之前：");
        System.out.println(Arrays.toString(a));
        Arrays.sort(a);
        System.out.println("排序之后：");
        System.out.println(Arrays.toString(a));
        System.out.println("数字37出现的位置："+Arrays.binarySearch(a, 37));
    }
}
```java
步骤 5 : 判断是否相同 
```java  
import java.util.Arrays;
public class HelloWorld{
    public static void main(String[] args) {
        int a[]=new int[]{1,2,3,5};
        int b[]=new int[]{1,2,3,5};
        System.out.println(Arrays.equals(a, b));
    }
}
```java
